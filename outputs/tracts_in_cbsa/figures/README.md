This folder stores the figures produced using pipeline outputs.

- `grid_lineplots` shows a grid with three metrics - Moran (P), Half Edge, Dissimilarity - side by side.
    - `_top10` plots show only 10 most populated CBSAs,
    - `_all_cbsa` show all of them and the yearly mean.
- `lineplots` show individual metric line plots.
- `metric_family_grids` groups metrics by family (e.g. all versions of Moran's I) and puts them on a grid.
- `other_highlights` (rename?) will store other plots.
